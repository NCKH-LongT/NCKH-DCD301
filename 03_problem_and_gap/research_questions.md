# Research Questions (RQs)

To successfully achieve the research objectives of this framework, the project focuses on answering four core Research Questions:

* [cite_start]**RQ1 (Data Quality):** How can the software system effectively detect, isolate, and filter out junk data or identify broken sensors in real time when the input Excel/CSV file is subjected to intentional error injection[cite: 9, 10]?
* [cite_start]**RQ2 (Retrieval-Augmented Generation - RAG):** What is the optimal approach to digitize physical agricultural manuals into a digital database and automatically search it to extract the correct technical paragraph when greenhouse incidents occur[cite: 13, 14]?
* [cite_start]**RQ3 (Agentic Decision Support):** How can an LLM-powered AI Agent be designed to synthesize cleaned real-time sensor data and retrieved technical knowledge to output optimal device control commands[cite: 19]?
* [cite_start]**RQ4 (Explainability & Trust):** How should the system output format be structured to gain the full trust of farmers by providing transparent reasoning and exact textual quotes from the original source documents[cite: 20]?
