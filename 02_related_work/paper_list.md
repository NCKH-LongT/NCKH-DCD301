# Paper List: FSM-based Password Door Lock System

## 1. Directly Related Papers (5 Papers)
*Focus: Digital electronic locks, FSM-based password system design.*

1. **Digital (Electronic) Locker**
   * **Authors:** N. Sharma, et al.
   * **Source:** 2020 First IEEE International Conference on Measurement, Instrumentation, Control and Automation (ICMICA)
   * **Focus:** Design of a 4-digit digital lock system using Verilog HDL and Finite State Machine, featuring a countdown timer and wrong password alarm.
   * **Link:** [https://ieeexplore.ieee.org/document/9242688/](https://ieeexplore.ieee.org/document/9242688/)

2. **FPGA-based assistive framework for smart home automation**
   * **Authors:** [Various authors]
   * **Source:** IEEE Xplore (2022)
   * **Focus:** Proposes a smart home framework where the password-protected door lock system is implemented via an FSM model.
   * **Link:** [https://ieeexplore.ieee.org/document/9845625/](https://ieeexplore.ieee.org/document/9845625/)

3. **FPGA Based System Login Security Lock Design Using Finite State Machine**
   * **Authors:** Saroch, et al.
   * **Source:** IOSR Journals (Indexed in Semantic Scholar)
   * **Focus:** Design of an automated secure login lock system using FSM based on FPGA and VHDL, unlocking only when the correct predefined sequence is entered.
   * **Link:** [https://www.semanticscholar.org/paper/FPGA-Based-System-Login-Security-Lock-Design-Using-Journals-Saroch/33f014663cc4503d921461014700b614fed36adb](https://www.semanticscholar.org/paper/FPGA-Based-System-Login-Security-Lock-Design-Using-Journals-Saroch/33f014663cc4503d921461014700b614fed36adb)

4. **Design of mealy finite-state machines with the transformation of object codes**
   * **Authors:** A. Barkalov, et al.
   * **Source:** EUDML (European Digital Mathematics Library)
   * **Focus:** Proposes a method to optimize logic circuits for Mealy state machine systems.
   * **Link:** [https://eudml.org/doc/207723](https://eudml.org/doc/207723)

5. **Synthesis of a Finite State Machine With Elementary Chains of States**
   * **Authors:** O. Barkalov, et al.
   * **Source:** 2022 IEEE 9th International Conference on Problems of Infocommunications, Science and Technology (PIC S&T)
   * **Focus:** Proposes a method and modified structure to synthesize a Combined FSM that generates output functions of both FSM types – Mealy and Moore.
   * **Link:** [https://ieeexplore.ieee.org/document/10238579/](https://ieeexplore.ieee.org/document/10238579/)

## 2. Methodology Papers (3 Papers)
*Focus: FSM methodology, Mealy/Moore comparison, and VHDL/Verilog coding.*

6. **Finite State Machine Design and VHDL Coding Techniques**
   * **Authors:** V. A. Pedroni
   * **Source:** 10th International Conference on DEVELOPMENT AND APPLICATION SYSTEMS
   * **Focus:** Fundamental issues in state machine design: state encoding, selecting Mealy or Moore output types, and VHDL coding techniques.
   * **Link:** [https://kolegite.com/EE_library/papers/HDL/Finite_state_machines_in_VHDL.pdf](https://kolegite.com/EE_library/papers/HDL/Finite_state_machines_in_VHDL.pdf)

7. **A Survey on Finite State Machine Encoding Techniques for Low Power Design**
   * **Authors:** [Various]
   * **Source:** IEEE Xplore 
   * **Focus:** Evaluates FSM encoding techniques (such as One-hot, Gray code) to minimize resource usage and power consumption for sequential circuits.

8. **State Minimization of Finite State Machines**
   * **Authors:** [Various]
   * **Source:** Springer / IEEE
   * **Focus:** Basic theory on minimizing the number of states in an FSM to reduce the complexity of the internal combinational logic circuit.

## 3. Application Domain Papers (2 Papers)
*Focus: Electronic lock systems, hardware security.*

9. **Practical Use of Datakey Tokens in Electronic Lock Systems**
   * **Authors:** Embedded.com
   * **Source:** Embedded Systems Publication (2025)
   * **Focus:** Survey of the practical architecture of an electronic lock system, and how the circuit interacts with the microcontroller to authenticate tokens/passwords.
   * **Link:** [https://www.embedded.com/practical-use-of-datakey-tokens/](https://www.embedded.com/practical-use-of-datakey-tokens/)

10. **Hardware-based Security for IoT Edge Devices: Electronic Access Control**
    * **Authors:** Y. Liu, et al.
    * **Source:** 2017 IEEE 60th International Midwest Symposium on Circuits and Systems (MWSCAS)
    * **Focus:** Research on hardware security in access control devices, threats to circuit design, and system protection solutions.
    * **Link:** [https://ieeexplore.ieee.org/document/8053155/](https://ieeexplore.ieee.org/document/8053155/)
