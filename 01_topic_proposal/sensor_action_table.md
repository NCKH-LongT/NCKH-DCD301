# Sensor Table, Meaning, and Related Actions

| Sensor | Meaning | Suggested Action |
|---|---|---|
| Temperature | Monitor the greenhouse ambient temperature | Turn fan on/off, alert when threshold exceeded |
| Humidity | Monitor air humidity | Turn fan on/off, alert for environmental adjustment |
| Soil moisture | Monitor soil moisture level | Automatic watering, alert when soil is too dry |
| Light | Monitor light intensity | Turn lights on/off, alert when light is insufficient |
