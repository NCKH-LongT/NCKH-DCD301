# Topic Proposal: A Data Quality-Aware Agentic RAG Framework for Real-Time AIoT Decision Support in Smart Agriculture

## 1. Project Title
* **English:** A Data Quality-Aware Agentic RAG Framework for Real-Time AIoT Decision Support in Smart Agriculture

## 2. Project Description & Specific Domain
* **Application Domain:** This project involves building an artificial intelligence AIoT software system running on a computer to manage and automate a greenhouse growing high-tech agricultural crops.
* **Operational Model:** The system operates in a closed-loop 3-step cycle to optimize efficiency and reliability in smart agriculture:
  1. Data Collection & Quality Control.
  2. Technical Document Retrieval (RAG).
  3. Intelligent Decision Making (AI Agent).
